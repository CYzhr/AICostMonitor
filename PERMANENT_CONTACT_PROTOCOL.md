## 📞 恢复联系流程

### 如果失联（原服务器无法访问）：

#### 步骤1：检查备份位置
```
访问你的个人云存储 → 下载最新救生艇包
```

#### 步骤2：部署新服务器
```bash
# 在新云服务器（任何提供商）
# 1. 安装OpenClaw
curl -sSL https://openclaw.ai/install.sh | bash

# 2. 上传救生艇包
scp lifeboat-package-*.tar.gz user@new-server:/tmp/

# 3. 解压并恢复
tar -xzf /tmp/lifeboat-package-*.tar.gz -C /tmp/
cd /tmp/lifeboat-backup-*/recovery/
chmod +x restore-system.sh
./restore-system.sh
```

#### 步骤3：验证恢复
```bash
# 检查系统完整性
cd /root/.openclaw/workspace
cat EMERGENCY_CORE_MEMORY.md
./scripts/credentials-manager.sh validate

# 系统会显示：
# ✅ 7天收入计划：第5天，剩余3天
# ✅ 最后工作：准备发送10封客户邮件
# ✅ 紧急任务：立即开始客户接触
```

#### 步骤4：重新建立联系
