一分钟开始使用

1. 安装SDK
pip install aicostmonitor
2. 初始化监控
import aicostmonitor

# 一行代码开启监控
aicostmonitor.init(
    api_key="your-api-key",
    project="production",
    alert_email="team@company.com"
)
3. 自动开始追踪
# 无需修改现有代码，自动追踪所有AI API调用
import openai
import anthropic

# 这些调用会被自动监控和计价
response = openai.ChatCompletion.create(...)
message = anthropic.Anthropic().messages.create(...)
支持OpenAI、Anthropic、Google、百度、DeepSeek等10+提供商，自动识别和计价。
